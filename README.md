**Rain Prediction using Artificial Neural Network (ANN)**
This project predicts whether it will rain tomorrow using weather parameters such as temperature, humidity, wind speed, precipitation, cloud cover, and pressure.

**Workflow**
Dataset loading
Data preprocessing
MinMax scaling
Train-test split
ANN model training
Model evaluation
Prediction using user input

**Model Architecture**
Input Layer → Dense(8, ReLU) → Output Layer (Sigmoid)

**Accuracy**
Model achieved approximately 99% accuracy on the dataset.

**Technologies Used**

Python
TensorFlow
Keras
Scikit-learn
Google Colab
